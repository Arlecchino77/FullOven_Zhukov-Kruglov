# FullOven_Zhukov-Kruglov

Автор: Круглов Александр, Жуков Вадим, группа 25ИС11Д.

## Описание
Учебный проект: сайт сети ресторанов (консольное приложение).

## План функций
- О ресторане
- Наши места на карте
- Открыть меню
- Закрыть меню
- Забронировать стол при помощи горяей линии
- Наши контакты
- Выход

## Стек
C# / .NET / Visual Studio

## Как запустить
1) Открыть solution в Visual Studio
2) Нажать F5 (Run)
3) В консоли появится меню
